# dwr

**NOTE:** Still very much a work in progress!

A hardware accelerated Wayland widget/bar creation library.

Implemented in Rust, custimizable using Lua. Inspired by AwesomeWM.

## Why?

Because why not?

There are *many* widget and bar creation tools out there, but none quite fit my needs.
